# Minesweeper

A build of the ever popular game minesweeper in Angular 20 and TypeScript. 

**Playable version:** 
Check out a playable version [here!](http://robinsoepboer.com/minesweeper)

**Requirements**

- node/npm
- Angular CLI
- yarn

## Running the Application

To run the application:

```
npm install
ng serve
```
